# basic electron site
---

## To run the project..
1. Have node js installed.
2. Install Electron Forge (install globally)
`npm install -g @electron-forge/cli`
3. Create the app in the directory that you want..
`npx create-electron-app my-electron-app`
4. change dir
`cd my-electron-app`
5. Run the app
`npm start`
